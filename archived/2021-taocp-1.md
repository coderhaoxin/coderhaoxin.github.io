### MMIX 基础

```
1 short (Wyde)  = 2 bytes
1 int   (Tetra) = 4 bytes
1 long  (Octa)  = 8 bytes
```

* 内存和寄存器
  - 内存 `M[0]`, `M[1]`, ..., `M[2^64 - 1]`
  - `M[x]` = 1 byte
  - 通用寄存器 `$0`, `$1`, ..., `$255`
  - `$x` = 1 long
  - 专用寄存器 `rA`, `rB`, ..., `rZ`, `rBB`, `rTT`, `rWW`, `rXX`, `rYY`, `rZZ`
  - 26 + 6 = 32, `r*` = 1 long

* 指令: 1 int
  - OP, X, Y, Z (1 byte, 1 byte, 1 byte, 1 byte)
  - OP: 操作码, `#20` means `ADD`
  - X, Y, Z: 操作数

```sh
ADD $X, $Y, $Z # 3 操作数
INCL $X, YZ # 2 操作数
JMP @+4*XYZ # 1 操作数
```

* **Load** 内存数据 -> 寄存器 `$X`

```sh
LDB $X, $Y, $Z
LDW $X, $Y, $Z
LDT $X, $Y, $Z
LDO $X, $Y, $Z
```

* **算术运算符**

```sh
ADD $X, $Y, $Z
SUB $X, $Y, $Z
MUL $X, $Y, $Z
DIV $X, $Y, $Z
```

* **浮点运算** `IEEE/ANSI 754`
  - 每一个 long (Octa) 8 bytes: `1 bit` 符号位, `11 bits` 指数位 E, `52 bits` 小数位 F

```sh
FADD $X, $Y, $Z
FSUB $X, $Y, $Z
FMUL $X, $Y, $Z
FDIV $X, $Y, $Z
```

* **跳转**
  - `JMP @+4*XYZ`
  - `@` 当前位置
  - `@+4*XYZ` 当前位置 `@` 之后的 XYZ 半字 (int, 4 bytes) 位置
  - `+`, 也可以是 `-`

* **转移**
  - `GO $X, $Y, $Z`
