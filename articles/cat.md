* 打架
  - 平时吧, 相互打闹, 大的欺负小的
  - 有一次, 不小心把小猫关在了卫生间
  - 大猫咪急的不得了, 不停的叫唤

* 葡萄的智商体现

```
某晚, 葡萄从柜子里拖出了一袋猫粮,
(比平常的伙食要好的那种, 偶尔给他加餐的)
从柜子里拖出, 拖到了沙发上, 咬坏了袋子,
把里面的猫粮倒到了沙发上, 吃了半袋子.
```

* 一边打架, 一边关怀

```
某天, 葡萄吃的撑了, 吐了一下
提子见状, 会在葡萄身边蹭一蹭
表示安慰

有的时候, 提子会在葡萄吃猫粮的时候
过去抢食
葡萄会伸爪子打一下
然后还是会让提子先吃
```
